**Projeto de CRUD para consolidar os estudos da primeira semana de kotlin;**  
O sistema consiste em um CRUD de séries, onde podemos cadastras, listar todas, listar por: id, título, gênero, editar, deletar;  
Tudo via console da aplicação, para fazer o projeto funcionar, basta abrir no inteliji ou android studio, colocar na versão 17 do java, e startar no arquivo main.  
Cada Atividade solicitada está no mesmo repositório, mas separados em branchs.  
**13/03/2026 → Atividade desenvolvida somente com Kotlin (Branch →  main)**
